# Ses Efektleri

Bu klasör uygulama ses efektlerini içerir.

## Gerekli Dosyalar

| Dosya | Açıklama | Önerilen Süre |
|-------|----------|---------------|
| `complete.mp3` | Görev tamamlama sesi | 0.5-1 saniye |
| `tap.mp3` | Buton tıklama sesi | 0.1-0.3 saniye |
| `celebration.mp3` | Kutlama sesi | 1-2 saniye |
| `notification.mp3` | Bildirim sesi | 0.5-1 saniye |

## Ücretsiz Ses Efektleri Kaynakları

1. **Freesound** - https://freesound.org
   - Ücretsiz, Creative Commons lisanslı sesler
   - "success sound", "complete", "ding" araması yapın

2. **Mixkit** - https://mixkit.co/free-sound-effects/
   - Tamamen ücretsiz, atıf gerektirmeyen sesler
   - "notification", "success", "achievement" kategorileri

3. **Zapsplat** - https://www.zapsplat.com
   - Ücretsiz üyelik ile indirme
   - Yüksek kaliteli UI ses efektleri

4. **Pixabay** - https://pixabay.com/sound-effects/
   - Ücretsiz, ticari kullanıma uygun
   - "ui", "interface", "success" araması

## Önerilen Ses Efektleri

### Görev Tamamlama (complete.mp3)
- Pozitif, başarı hissi veren
- Örnek: Zil sesi, "ding", level up sesi

### Buton Tıklama (tap.mp3)
- Kısa ve hafif
- Örnek: Yumuşak tık sesi

### Kutlama (celebration.mp3)
- Neşeli, enerji verici
- Örnek: Konfeti, alkış, fanfare

### Bildirim (notification.mp3)
- Dikkat çekici ama rahatsız edici değil
- Örnek: Soft ping, chime

## Dosya Formatı

- **Format:** MP3 (önerilen) veya WAV
- **Bit rate:** 128-320 kbps
- **Sample rate:** 44100 Hz
- **Dosya boyutu:** Mümkün olduğunca küçük tutun (< 100KB)
